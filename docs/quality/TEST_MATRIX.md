## 按变更领域要求的测试套件

- 领域/指标/因子：单元测试与属性测试。
- 供应商/存储：使用录制或假响应的契约测试与集成测试。
- 规则/执行/回测：单元、场景和确定性回归测试。
- GUI/ViewModel：使用pytest-qt测试状态、取消、代次和错误状态。
- 策略/模型：时点数据集检查、滚动前推评估和模型卡产物。
- 盈利证据：固定交易路径成本压力、无交易折进入滚动分母、目标波动仓位不使用杠杆、交易成本按实际暴露计算、数据快照进入报告校验和、数据截断/空响应不得静默通过，并复跑至少两组十标的池。
- 打包：干净环境安装/启动冒烟测试，不得嵌入凭据。
