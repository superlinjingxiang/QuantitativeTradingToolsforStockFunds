# 验收条件

## 产品验收

| ID | 条件 | 证据 |
|---|---|---|
| AC-01 | 输入有效代码或名称能够返回正确候选并完成证券选择。 | GUI/集成测试 |
| AC-02 | 图表、实时行情、策略、预测和信号共享同一个选中证券及同一时点上下文。 | 证券切换代次端到端测试 |
| AC-03 | 图表支持要求的周期、平移/缩放/悬停、信号标记和预测区间。 | GUI测试和视觉证据 |
| AC-04 | 策略名称、版本、周期、逻辑、适用条件和失效条件均可见。 | GUI测试 |
| AC-05 | 预测使用概率和区间，不包含保证收益的表述。 | Schema/内容测试 |
| AC-06 | 陈旧或非法数据会阻止新信号，并明确显示原因。 | 故障注入测试 |
| AC-07 | 不存在已知未来数据泄漏，成交时间可确定复现。 | 回测时序测试套件 |
| AC-08 | 建模成本、滑点、涨跌停、停牌、可卖数量和按日期生效的规则。 | 场景测试 |
| AC-09 | 相同快照、配置、模型、规则和随机种子产生完全一致的结果。 | 确定性回归测试 |
| AC-10 | 每个信号可追溯到数据时间/快照、策略/模型/规则版本和解释。 | 审计契约测试 |
| AC-11 | A股、场内基金和场外基金不得共用错误的数据或执行语义。 | 领域契约测试 |
| AC-12 | 策略在通过最终样本外和模拟交易门槛前，不得标记为生产可用。 | 生命周期测试/评审证据 |

## 非功能验收

| ID | 条件 |
|---|---|
| NFR-01 | 在约定夹具和硬件上，本地搜索P95小于300毫秒。 |
| NFR-02 | 缓存图表首次绘制目标不超过1秒；远程补全异步执行。 |
| NFR-03 | Qt主线程不得出现超过100毫秒的阻塞操作。 |
| NFR-04 | 实时断线必须可见，并使用有上限的指数退避重连。 |
| NFR-05 | 长时间回测显示进度并支持取消；目标硬件上记录基准。 |
| NFR-06 | 后台任务可取消、在安全场景下可重试，并保持幂等。 |
| NFR-07 | 异常退出后可恢复用户设置、自选和模拟组合。 |
| NFR-08 | 随机性、版本和校验和足以复现计算结果。 |

## 研究准入门槛

策略不能仅因总收益为正而通过。必须提供扣除成本后的最终样本外结果、回撤/尾部风险限制、基准比较、概率校准、样本充分性、不同市场状态和年份的稳健性、参数敏感性、容量/流动性，以及模拟交易偏差处于预先声明容差内的证据。

## 下一阶段策略决策中枢验收门槛（TASK-026）

这些门槛用于后续策略决策中枢MVP，不计入当前已完成发布审计的AC-01至AC-12。

| ID | 条件 | 证据 |
|---|---|---|
| SDA-001 | 任一当前标的可生成统一决策报告，覆盖行情、策略、预测、风险、回测、模拟盘和审计摘要。 | DecisionReport单元/集成测试 |
| SDA-002 | 最终建议必须限定为买入候选、持有、减仓、卖出、观察或ABSTAIN，并展示置信度、仓位上限、有效期和失效条件。 | 策略决策测试和GUI测试 |
| SDA-003 | 数据、样本外、成本压力、风险预算、概率校准或模拟盘证据不足时，必须降级为观察或ABSTAIN。 | 门禁故障注入测试 |
| SDA-004 | 决策报告必须展示扣除成本后的净收益、最大回撤、基准比较、交易次数和稳定性说明。 | 回测证据汇总测试 |
| SDA-005 | GUI必须在当前标的上下文中展示策略建议和赚钱证据，同时保留不保证盈利的边界说明。 | GUI内容测试 |
| SDA-006 | 决策中枢不得提交真实订单，只能输出未来API执行候选状态和必要风控字段。 | 无真实下单路径测试 |

## 下一阶段盈利验证深化门槛（TASK-027）

TASK-027优先回答“策略到底能不能赚钱、回测是否可信、模拟盘是否能验证”。Electron/Web前端重构不得替代或降低这些门槛。

| ID | 条件 | 证据 |
|---|---|---|
| EPV-001 | 每个候选策略必须有固定训练、验证、最终样本外和滚动前推切分，并记录数据快照、证券池和规则版本。 | 样本切分/无泄漏测试 |
| EPV-002 | 回测报告必须同时给出扣除成本后的净收益、最大回撤、风险调整收益、基准比较、交易次数、换手率和年份/市场状态拆解。 | 回测报告回归测试 |
| EPV-003 | 必须运行成本、滑点、流动性容量、涨跌停/停牌和参数敏感性压力测试；压力后失效的策略不得进入执行候选。 | 压力测试与敏感性测试 |
| EPV-004 | 必须提供过拟合和多重检验诊断，包括参数稳定性、样本量下限、校准误差和策略选择偏差说明。 | 过拟合诊断测试/模型卡 |
| EPV-005 | 模拟盘必须记录信号时间、可得报价、模拟成交、滑点偏差、漏单/重复信号、数据延迟和异常恢复；偏差超限时自动降级。 | 模拟盘偏差测试 |
| EPV-006 | DecisionHub只能使用通过EPV门槛的证据提升执行候选等级；证据缺失、过期或失败时输出WATCH或ABSTAIN。 | DecisionHub集成门禁测试 |
