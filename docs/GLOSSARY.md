# 术语表

| 术语 | 定义 |
|---|---|
| OHLC/K线 | 某一时间区间的开盘价、最高价、最低价和收盘价。 |
| T+ / 回转交易 | 必须结合资产类型和规则语境解释的时间、确认和可卖性约束。 |
| 时点数据（Point-in-time data） | 按历史决策时刻实际可观察状态保存的数据。 |
| 未来数据泄漏（Look-ahead leakage） | 使用了模拟决策时点尚不可获得的信息或价格。 |
| 幸存者偏差（Survivorship bias） | 排除失败、退市或历史成分，只使用当前仍存在的证券。 |
| 正期望值（Positive expectancy） | 扣除成本后的长期平均结果为正，不代表每笔交易都盈利。 |
| 最大回撤（Maximum drawdown） | 组合净值从某一峰值到随后谷值的最大跌幅。 |
| 概率校准（Calibration） | 模型给出的概率与长期实际发生频率之间的一致程度。 |
| MAE | 持仓期间最大不利变动（Maximum Adverse Excursion）。 |
| ABSTAIN | 因证据不足或不安全，系统主动拒绝给出交易建议。 |
| 数据快照（Data snapshot） | 对结果所使用全部数据进行不可变、可版本化的标识。 |
| 规则版本（Rule version） | 按生效日期管理的一组交易与执行约束。 |
| 正式净值（Official NAV） | 按基金规则正式披露的净值，与盘中估算值不同。 |
| 模拟交易（Paper trading） | 使用实时数据驱动的模拟订单和持仓，不提交真实订单。 |
