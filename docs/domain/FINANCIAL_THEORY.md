# 国际通用金融理论基线

## 目的

使用国际认可的金融学、计量经济学、统计学和风险管理框架提出可检验假设。理论被发表或广泛接受，并不代表其可交易实现一定在中国市场有效；所有实现都必须使用时点中国市场数据，并在扣除成本后完成验证。

## 接受的理论类别

| 理论/框架 | 系统用途 | 必须避免的误用 |
|---|---|---|
| 现代投资组合理论 | 分散化、协方差、有效风险配置 | 把不稳定的历史协方差当作确定真值 |
| CAPM与风险调整绩效 | Beta、Alpha归因、基准感知评估 | 把CAPM当作次日价格预测模型 |
| 多因子资产定价 | 价值、质量、规模、投资、盈利、低风险等假设 | 假定因子溢价会跨市场、跨时期原样复制 |
| 动量 | 横截面/绝对动量与趋势假设 | 在没有风险和流动性控制时外推短期暴涨 |
| 均值回归 | 相对价值和过度反应假设 | 仅因价格下跌就买入 |
| 行为金融 | 反应不足、过度反应、从众和套利限制 | 把叙事直接转成未经检验的信号 |
| ARCH/GARCH与波动率模型 | 时变波动率和预测区间 | 把波动率预测误当作方向预测 |
| Black-Litterman/风险预算 | 结合均衡收益、主观看法和组合约束 | 把主观看法伪装成客观预测 |
| 概率校准 | 产生可解释的事件概率 | 把未经校准的分类器分数当作概率 |
| 时间序列验证 | 检验按时间推广能力和模型漂移 | 对重叠金融标签使用随机交叉验证 |

## 实施规则

- 每个因子必须记录定义、经济或统计理由、数据可用时间、预期方向、标准化方法、证券池、再平衡频率和失效条件。
- 先建立简单基准，再采用复杂模型。
- 同时评估预测指标和扣除成本后的真实组合指标。
- 报告置信区间和参数敏感性。
- 分离假设提出、模型选择和最终未触碰测试集。
- 技术指标和K线特征只能作为输入，不是独立理论。
- 不得把特征重要性或相关关系描述成因果关系。

## 组合优先解释

单只证券评分高，不代表应配置大仓位。最终仓位必须考虑波动率、下行风险、流动性、相关性、行业/主题集中度、市场状态、现有持仓和回撤预算。

## 经典研究参考

- Markowitz（1952）：投资组合选择。
- Sharpe/Lintner：CAPM与系统性风险。
- Fama-French：多因子资产定价。
- Jegadeesh-Titman：横截面动量。
- Engle：ARCH与时变波动率。
- Black-Litterman：均衡收益与投资者观点。

这些研究用于确定研究方向，不等于可直接上线的生产策略。
